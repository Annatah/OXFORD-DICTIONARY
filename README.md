
## Dictionary App 
# About:

This project is a simple dictionary application built with JavaScript and HTML. It allows users to search for word definitions and pronunciations.

## Features:

Fetches word definitions and pronunciations from the "Dictionary API" (https://dictionaryapi.dev/).
Displays definitions, part of speech, and phonetic spelling.
Plays audio pronunciation for the searched word (US English accent).
Handles potential errors like misspelled words or API issues.

## Technologies Used:

>HTML: For the basic structure of the web page.
>CSS (not included in this example): Used for styling the user interface.
>JavaScript: Fetches data from the API, parses the response, displays information, and handles user interactions.

## Running the Application:

Save the provided JavaScript code (along with any necessary HTML and CSS files) in a folder.
Open the HTML file in your web browser.
Enter a word in the search bar and press Enter or click the search icon.
Code Breakdown:

1. Variables:

input: References the search input field in the HTML.
result: References the element where the definition is displayed.
sound: References the audio element for playing pronunciation.
